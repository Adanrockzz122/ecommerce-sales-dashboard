# E-Commerce Sales Dashboard

Power BI dashboard analyzing 16M+ e-commerce revenue data.

## Key Insights

* Revenue peaks in November indicating seasonal demand
* Average Order Value (AOV) ~160
* Top cities contribute significantly to total revenue
